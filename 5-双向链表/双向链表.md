# 双向链表

## 单链表单另一个缺陷

- 单向性
  - 只能从头节点开始访问链表中的数据元素
- 缺陷
  - 如果需要逆向访问链表中数据元素将极其低效

```c++
int main()
{
    LinkList<int> l;

    // O(n)
    for(int i = 0; i < 5; i++)
    {
        l.insert(0, i);
    }

    // O(n^2)
    for(int i = l.length() - 1; i >= 0; i--)
    {
        cout << l.get(i) << endl;
    }
}
```

## 新的线性表

- 设计思路：
  - 在单链表中增加一个指针pre
  - 用于指向当前节点的前驱节点
